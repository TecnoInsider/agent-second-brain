# Маркетинговые хуки для схем вязания

## Суть идеи

Создать систему продажи идеи "вязание своими руками" через социальные сети для магазина жены на Etsy.

## Задача

Сгенерировать 10-20 маркетинговых крючков (хуков), которые:
- Объясняют почему выгодно вязать игрушки самостоятельно
- Создают желание купить схему и связать подарок
- Работают для постов и видеороликов

## Целевая аудитория

Люди, которые хотят:
- Дарить уникальные подарки
- Иметь хобби своими руками
- Создавать персонализированные игрушки для детей/близких
- Экономить на покупке готовых игрушек

## Форматы использования

1. **Посты в соцсетях** — короткие продающие тексты
2. **Видеоролики** — сценарии для съёмки процесса
3. **Рекламные кампании** — тексты для таргета

## Следующие шаги

- [ ] Провести исследование аудитории (что мотивирует к вязанию)
- [ ] Выписать 10-20 хуков
- [ ] Создать шаблоны постов
- [ ] Написать 2-3 сценария для видео
