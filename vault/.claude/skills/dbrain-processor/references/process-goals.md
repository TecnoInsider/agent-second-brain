# Действия вместо результатов

## Концепция

**Цели-результаты** фокусируются на ТОМ, чего хочешь достичь:
- "Заработать $3K/мес"
- "Запустить BaZi-сервис"
- "Переехать в Валенсию"

**Цели-действия** фокусируются на ТОМ, как это делаешь:
- "1 BaZi-ролик в день"
- "2 сессии по 2 часа на калорий-бот в неделю"
- "Написать 1 человеку в день с предложением бесплатной расшифровки"

## Почему действия работают лучше результатов

Based on research (meta-analysis of 27 experiments in sports):

1. **Контроль** — Ты контролируешь действия, не результат. «Продать расшифровку» создаёт тревогу; «написать 1 человеку» — понятно и выполнимо.
2. **Уверенность** — Процессные цели превращают большую задачу в маленькие шаги, повышая самоэффективность.
3. **Фокус** — Ежедневная ясность: что делать, а не абстрактное «надо бы...».

**Особенно важно для Вадима:** Склонность к перфекционизму (баг Петуха) делает цели-результаты токсичными — «запустить продукт» вызывает паралич «ещё не готово». Цели-действия обходят этот баг: ты просто делаешь шаг, не оценивая готовность.

## Как трансформировать

| Цель-результат (ПЛОХО) | Цель-действие (ХОРОШО) |
|----------------------|----------------------|
| "Запустить BaZi-сервис" | "2 часа работы над дашбордом каждое утро" |
| "Набрать 1000 подписчиков" | "2–3 AI-ролика + 1 YouTube видео в неделю" |
| "Начать зарабатывать на BaZi" | "1 бесплатная расшифровка в неделю + снять видео про процесс" |
| "Закрыть кредитки" | "Каждый платёжный день — оплатить минимум по всем картам" |
| "Пройти курс Анарбаевой" | "1 урок курса 3 раза в неделю" |
| "Выучить BaZi" | "1 обучающий ролик в день" |
| "Добить калорий-бот" | "2 сессии по 2 часа с помощницей в неделю до 28 февраля" |
| "Переехать в Валенсию" | "1 час в неделю: поиск квартир / расчёт бюджета / чеклист документов" |

## Паттерны для целей-действий

### Временные блоки
Регулярное выделенное время для глубокой работы:
- "2 часа работы над BaZi-дашбордом каждое утро 8:00–10:00"
- "1 час BaZi-обучение после обеда"
- "30 мин контент (пост/ролик) перед обедом"

### Регулярные обязательства
Постоянный ритм действий:
- "1 BaZi-ролик в день (обучение)"
- "2–3 AI-ролика в неделю (контент)"
- "1 бесплатная расшифровка в неделю (кейсы + сарафан)"
- "1 YouTube видео в неделю"

### Контрольные точки
Вехи для проверки прогресса:
- "Воскресенье: 15 мин недельный обзор"
- "Конец месяца: обзор прогресса по целям"
- "Показать черновик оффера 1 человеку до пятницы"

### Серии
Ежедневные привычки:
- "Зарядка + контрастный душ каждое утро"
- "Отбой до 23:00"
- "1 BaZi-ролик в день — не пропускать"

## При обработке голосовых записей

Если пользователь говорит цель-результат, трансформируй в цель-действие:

```
Пользователь: "Надо начать продавать BaZi-консультации"

ПЛОХО: Создать задачу "Начать продавать BaZi-консультации" p2
→ Размыто, вызывает тревогу, непонятно что делать

ХОРОШО: Создать задачи:
1. "Написать 1 знакомому с предложением бесплатной расшифровки" p3 due:tomorrow
2. "Набросать черновик оффера: что входит + цена (30 мин)" p3 due:friday
→ Конкретно, выполнимо, контролируемо
```

```
Пользователь:"Хочу закончить калорий-бот"

ПЛОХО:Create task "Закончить калорий-бот" p2
→ Слишком большое, парализует

ХОРОШО:Create task "Сессия с помощницей: демо-режим часть 1 (2 часа)" p2 due:tomorrow
→ Один конкретный шаг, ограничен по времени
```

```
Пользователь:"Нужно больше подписчиков на канале"

ПЛОХО:Create task "Набрать подписчиков" p3
→ Outcome, не контролируется

ХОРОШО:Create task "Записать быстрый AI-ролик с экрана (10 мин)" p3 due:today
→ Действие, которое 100% в твоём контроле
```

## Для еженедельного планирования

При создании действий-обязательств для ОДНОГО главного дела:

1. **Спроси:** Какие ежедневные/еженедельные привычки сделают это неизбежным?
2. **Ограничь:** 3–5 действий-обязательств максимум (правило фокуса!)
3. **Приоритизируй:** Временные блоки > Регулярность > Контрольные точки
4. **Создай:** Повторяющиеся задачи в Todoist с подходящими паттернами
5. **Проверь:** Не создаёт ли это растекание? (баг Свиньи) Лучше 3 действия, которые выполняются, чем 7 которые забрасываются.

## Паттерны повторяющихся задач

Соответствие целей-действий и dueString в Todoist:

| Действие | dueString |
|---------|-----------|
| "каждое утро" | "every day at 7:30" |
| "каждый день" | "every day" |
| "в рабочие дни" | "every weekday" |
| "3 раза в неделю" | "every monday, wednesday, friday" |
| "раз в неделю" | "every week" |
| "к пятнице" | "friday" (one-time) |
| "2 раза в неделю" | "every tuesday, thursday" |

## Анти-паттерны

Не создавай цели-действия, которые:
- Слишком расплывчаты: "Работать над проектом" (сколько? когда? над чем конкретно?)
- Слишком амбициозны: "4 часа глубокой работы 5 дней в неделю" (нереалистично → бросишь через неделю)
- Неотслеживаемы: "Думать о стратегии" (не измеримо)
- Слишком много: 7+ целей-действий одновременно (растекание → баг Свиньи)

Хорошие цели-действия:
- **Конкретны** в действии (что именно делаю)
- **Ограничены** по времени/частоте (сколько минут, сколько раз)
- **Отслеживаемы** (сделал / не сделал — бинарно)
- **Посильны** (правило 80%: лучше делать 80% каждый день, чем 100% раз в месяц)

## Источники

Основано на: Мета-анализ 27 экспериментов в спорте показал, что цели-действия значительно превосходят цели-результаты.

См. также: "The Success Equation" by Mauboussin — Результат = Навык + Удача. Фокус на навыке (действиях) — единственная контролируемая стратегия.
